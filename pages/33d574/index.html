    еңЁиҝҗиЎҢејҖеҸ‘жңҚеҠЎnpm run devжҲ–жү“еҢ…npm run buildж—¶дё»йўҳеҶ…йғЁе°ҶдјҡжҢүз…§зӣ®еҪ•зәҰе®ҡиҮӘеҠЁз”ҹжҲҗдёҖдёӘз»“жһ„еҢ–зҡ„дҫ§иҫ№ж ҸгҖҒзӣ®еҪ•йЎөгҖҒйқўеҢ…еұ‘зӯүж•°жҚ®гҖӮ

    еңЁжәҗзӣ®еҪ•пјҲдёҖиҲ¬жҳҜdocsпјүж–Үд»¶еӨ№дёӯпјҢйҷӨдәҶ.vuepressгҖҒ@pagesгҖҒ_postsгҖҒindex.md жҲ– README.mdд№ӢеӨ–зҡ„ж–Үд»¶еӨ№е°Ҷдјҡдёәе…¶з”ҹжҲҗеҜ№еә”зҡ„дҫ§иҫ№ж ҸгҖӮз”ҹжҲҗзҡ„йЎәеәҸеҸ–иҮӘеәҸеҸ·пјҢж ҮйўҳеҸ–иҮӘж–Үд»¶пјҲеӨ№пјүеҗҚз§°гҖӮ

    # е‘ҪеҗҚзәҰе®ҡ

    • ж— и®әжҳҜж–Үд»¶иҝҳжҳҜж–Үд»¶еӨ№пјҢиҜ·дёәе…¶еҗҚз§°ж·»еҠ дёҠжӯЈзЎ®зҡ„жӯЈж•ҙж•°еәҸеҸ·е’Ң.пјҢд»Һ00жҲ–01ејҖе§ӢзҙҜи®ЎпјҢеҰӮ01.ж–Үд»¶еӨ№гҖҒ02.ж–Үд»¶.mdпјҢжҲ‘д»¬е°ҶдјҡжҢүз…§еәҸеҸ·зҡ„йЎәеәҸжқҘеҶіе®ҡе…¶еңЁдҫ§иҫ№ж ҸеҪ“дёӯзҡ„йЎәеәҸгҖӮ
    • еҗҢдёҖзә§еҲ«зӣ®еҪ•еҲ«еҶ…еҚідҪҝеҸӘжңүдёҖдёӘж–Үд»¶жҲ–ж–Үд»¶еӨ№д№ҹиҰҒдёәе…¶еҠ дёҠеәҸеҸ·гҖӮ
    • ж–Үд»¶жҲ–ж–Үд»¶еӨ№еҗҚз§°дёӯй—ҙдёҚиғҪеҮәзҺ°еӨҡдҪҷзҡ„зӮ№.пјҢеҰӮ01.жҲ‘жҳҜ.еҗҚз§°.mdдёӯй—ҙеҮәзҺ°.е°ҶдјҡеҜјиҮҙи§Јжһҗй”ҷиҜҜгҖӮ

    жҸҗзӨә

    еәҸеҸ·еҸӘжҳҜз”ЁдәҺеҶіе®ҡе…ҲеҗҺйЎәеәҸпјҢе№¶дёҚдёҖе®ҡйңҖиҰҒиҝһзқҖпјҢеҰӮ01гҖҒ02гҖҒ03...пјҢе®һйҷ…е·ҘдҪңдёӯеҸҜиғҪдјҡеңЁдёӨдёӘж–Үз« дёӯй—ҙжҸ’е…ҘдёҖзҜҮж–°зҡ„ж–Үз« пјҢеӣ жӯӨдёәдәҶж–№дҫҝеҸҜд»ҘйҮҮз”Ёй—ҙйҡ”еәҸеҸ·10гҖҒ20гҖҒ30...пјҢеҗҺйқўеҰӮжһңйңҖиҰҒеңЁ10е’Ң20дёӯй—ҙжҸ’е…ҘдёҖзҜҮж–°ж–Үз« пјҢеҸҜд»Ҙз»ҷе®ҡеәҸеҸ·15гҖӮ

    # зә§еҲ«иҜҙжҳҺ

    жәҗзӣ®еҪ•пјҲдёҖиҲ¬жҳҜdocsпјүеә•дёӢзҡ„зә§еҲ«зҺ°еңЁжҲ‘д»¬з§°д№ӢдёәдёҖзә§зӣ®еҪ•пјҢдёҖзә§зӣ®еҪ•зҡ„дёӢдёҖзә§дёәдәҢзә§зӣ®еҪ•пјҢд»ҘжӯӨзұ»жҺЁпјҢжңҖеӨҡеҲ°еӣӣзә§зӣ®еҪ•гҖӮ

    • дёҖзә§зӣ®еҪ•
      1. .vuepressгҖҒ@pagesгҖҒ_postsгҖҒindex.md жҲ– README.md иҝҷдәӣж–Үд»¶(ж–Үд»¶еӨ№)дёҚеҸӮдёҺж•°жҚ®з”ҹжҲҗгҖӮ
      2. еәҸеҸ·йқһеҝ…йЎ»гҖӮпјҲеҰӮдёҖдәӣдё“ж ҸпјҢеҸҜд»ҘдёҚз”ЁеәҸеҸ·)
    • дәҢзә§зӣ®еҪ•
      1. иҜҘзә§еҲ«дёӢеҸҜд»ҘеҗҢж—¶ж”ҫж–Үд»¶еӨ№е’Ң.mdж–Үд»¶пјҢдҪҶжҳҜдёӨиҖ…еәҸеҸ·иҰҒиҝһиҙҜпјҲеҸӮиҖғдёӢйқўзҡ„дҫӢеӯҗдёӯзҡ„е…¶д»–пјүгҖӮ
      2. еҝ…йЎ»жңүеәҸеҸ·
    • дёүзә§зӣ®еҪ•
      • (еҗҢдёҠ)
    • еӣӣзә§зӣ®еҪ• v1.6.0 +
      1. иҜҘзә§еҲ«дёӢеҸӘиғҪж”ҫ.mdж–Үд»¶гҖӮ
      2. еҝ…йЎ»жңүеәҸеҸ·
